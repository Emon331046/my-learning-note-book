# postgres-CLI-command
---

## start postgres

```shell
$ sudo -u postgres psql postgres
```

---

## set new password

```
$ \password
```
---
## show all database
```
$ \l
```
---
 
 ## connect to the database
 ```
 $ \c database_name
 ```
 ---
##  show all tables in a database
```
$ \dt
```
---
##  quit
```
$ \q
```
